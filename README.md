# Reactor

Experiment making an isomorphic app using React.

## Installation

    npm run first-run

And start it:

    npm start

And head to [localhost:3111](localhost:3111).

## What it does

- Renders app from server, continues app on client
- Routing
- Pages fetching from API and receiving state
- pushState navigation
